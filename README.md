# cra-template-rrrs

The template for Create React App

## Start Template

### yarn

```
yarn create react-app my-app cra-template-rrrs
```

### npm

```
npm init react-app my-app cra-template-rrrs
```

## template support

- typescript 4.4.x
- styled-components
- react-router
- react-query
